## Roll Dice! üé≤
Use `/roll` to roll dice. `/roll` has the following subcommands. 
<br>
<br>
‚Ä¢ `/roll d6` lets users roll 1-10 D6s. This command has one required field called *amount*, which is where the user places the number of D6s they wish to roll. If a non-integer or number greater than 10 is placed in *amount*, Vista will respond with an error. <br>
‚Ä¢ `/roll d100` lets users roll 1-10 D100s. Like `/roll D6`, this command has one required field called *amount*, and sends an error if a non-integer or number greater than 10 is inputed. <br>
‚Ä¢ `/roll ability` lets users roll on the Ability table to get a random ability. <br>
‚Ä¢ `/roll aspect` lets users generate a random NPC Aspect on the Aspect table. <br>
